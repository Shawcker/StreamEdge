Module responsible for handling conditionals.

See also: functions.

Description

Module responsible for handling conditionals. Executes a given function when the given conditional (predicate) is verified. After function executes, waits for predicate to be false and then executes it again. It works like a cycle, where the function is executed when predicate changes value.
